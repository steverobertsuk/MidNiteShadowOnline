A new update has been released for [**STO Info**](https://startrekonline.info/) with a number of improvements to the dashboard and behind-the-scenes systems.

### 🚀 New Features

- **Character Statistics** - The dashboard now adds statistics and supports filtering for character statistics, making it easier to view and analyse your data.
- **Automated Release Notes** - Release notes are now generated automatically as part of the CI pipeline.
- **Improved Contributors Automation** - Contributor recognition is now handled automatically.

### 🐛 Fixes & Improvements

- UI alignment improvements on the home page
- CI workflow fixes and reliability improvements
- Updated contributor badge links

### ⚙️ Maintenance

- Dependency updates across Angular and tooling
- Security dependency overrides added
- CodeQL and CI workflow improvements

This release continues the focus on improving the **dashboard experience while strengthening the platform’s reliability and security**.
