A new update has been released for the **STO Info community project**, introducing a **refactored user interface, moving the site toward a more LCARS-inspired design**.

The goal of this change is to bring the interface closer to the **Star Trek aesthetic** while continuing to expand features and improve usability.

🌐 You can explore the site here: [https://startrekonline.info](https://startrekonline.info)

For anyone unfamiliar with the project, **STO Info** is a free community resource that helps Star Trek Online players track game information, statistics, and database changes.

**Release notes:**

- Frontend v1.0.162: [https://github.com/sto-info-app/sto-info-frontend/releases/tag/v1.0.162](https://github.com/sto-info-app/sto-info-frontend/releases/tag/v1.0.162)
- Backend v1.0.141: [https://github.com/sto-info-app/sto-info-backend/releases/tag/v1.0.141](https://github.com/sto-info-app/sto-info-backend/releases/tag/v1.0.141)

The project is open source, and community feedback is always welcome.
