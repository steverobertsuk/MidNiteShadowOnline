## Swap Token Positions Overview

Swap Token Positions is a Roll20 API utility that instantly exchanges the positions of exactly two selected tokens.

It is aimed at tables that frequently use forced movement, teleports, repositioning powers, encounter mechanics, or Game Master adjudication that requires two tokens to trade places cleanly.

Instead of dragging tokens manually and risking overlap, misalignment, or accidental layer changes, the script validates the selection and performs the swap automatically.

---

## Key Features

### Two-Token Position Swapping

Select two tokens, run the command, and the script exchanges their positions.

### Clean Movement Handling

Swap Token Positions is designed to preserve important token properties during the move, helping avoid common manual drag-and-drop mistakes.

### Selection Validation

The script provides clear feedback when the selection is invalid, incomplete, or contains the wrong number of tokens.

### Fast Encounter Adjudication

The utility is useful for movement powers, magical swaps, teleports, monster abilities, traps, puzzles, and quick GM repositioning.

---

## Why Use It?

Small token movement tasks can slow down play when they need to be handled carefully.

Swap Token Positions provides a quick, reliable way to exchange two token positions without interrupting the pace of the encounter.

---

## Compatibility

Swap Token Positions is designed for Roll20 API games and is intended to work cleanly with standard token movement workflows.
